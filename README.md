Things that I learned in Lesson 9 :
1. Switch case
For example : 
let John = 34;
switch (John) {
    case 33: 
    console.log("You are a little bit younger");
       break;
    case 34:
    console.log("You are well aged");
       break;
    case 35:
    console.log("You are a little bit older");
       break;
    default: 
       console.log("Come later"); 
}
